## Part 1 — 每日晨间流程

### 1.1 一键生成多赛道晨报

每天开盘前，用一条指令获取所有追踪赛道的隔夜动态：

```
帮我生成今日投研晨报，追踪赛道：半导体设备、光通信、先进封装、AI算力。
重点关注：英伟达GTC动态、国产替代进展、美国出口管制变化。
只看最近1天的内容。
```

> **技巧**：`days=1` 只拉前一日新增，避免信息重复。周一改为 `days=3` 补上周末。

### 1.2 快速扫描隔夜券商观点变化

晨报之外，单独拉取关注个股的最新券商评级变化：

```
搜索最近2天关于北方华创、中微公司、拓荆科技的券商研报和分析师观点
```

> **技巧**：用 `resource_types: ["券商研报", "分析师观点"]` 过滤，排除公众号噪音。

### 1.3 盘前数据确认

用指标查询验证关键数据点：

```
查询2025年全球半导体设备出货额按季度数据
```

```
查询中芯国际最近4个季度营收和资本开支
```

### 1.4 推荐晨间 Prompt 模板

```
你是我的投研助理。请完成以下晨间任务：

1. 调用 gangtise_daily_report，赛道：半导体设备、光通信、先进封装，days=1
2. 用 gangtise_knowledge_search 搜索"ASML 英伟达 台积电"最新公告，限3天
3. 用 gangtise_indicator 查询"全球AI服务器出货量 2025"

请将结果整合为一份简洁的晨间 Brief，按以下格式：
- 隔夜重大事件（3条以内）
- 各赛道核心变化
- 今日需要关注的催化剂
- 风险提示
```

## Part 2 — 初次覆盖一个新行业

当你被分配覆盖一个全新的细分赛道时（例如：光通信），以下是从零到一的系统性研究路径。

### 2.1 第一步：建立行业认知框架（30 分钟）

用批量检索同时发起多维度查询：

```
批量查询以下问题：
1. 光通信行业产业链全景图 上中下游
2. 英伟达GPU集群网络架构 交换机 光模块规格
3. 800G 1.6T光模块技术路线 硅光 EML
4. 光模块行业竞争格局 市占率
5. 光通信国产替代进度 光芯片
```

### 2.2 第二步：量化行业规模与增速

```
查询以下指标：
- 全球光模块市场规模 2023 2024 2025
- 中国数据中心光模块出货量
- 800G光模块 ASP 均价趋势
```

### 2.3 第三步：AI Agent 产业链深度拆解

```
使用产业链分析Agent，研究问题：
光通信产业链深度分析——从光芯片到光模块到交换机到系统集成，
每个环节的技术壁垒、竞争格局、国产化率分别是什么？
哪些环节的投资价值最高？
```

### 2.4 第四步：生成调研提纲

在建立框架后，为即将到来的公司调研准备问题：

```
使用调研提纲Agent，为中际旭创调研生成提纲。
重点关注：800G/1.6T产能爬坡节奏、海外大客户份额变化、
硅光vs EML路线选择、毛利率走势。
```

### 2.5 第五步：多轮对话深入追问

对于复杂问题，先创建会话再逐步追问：

```
第一轮：创建一个研究会话
第二轮（带 group_id）：分析英伟达 Blackwell 平台对光模块需求的拉动有多大？
第三轮（带 group_id）：在这个基础上，对比分析中际旭创和新易盛谁更受益？
第四轮（带 group_id）：如果以太网替代 InfiniBand 加速，对上述判断有什么影响？
```

> **技巧**：多轮对话会保持上下文，后续追问不需要重复背景信息，Agent 的回答会更有连贯性和深度。

## Part 4 — 投资决策支持

### 4.1 构建投资逻辑

当你准备向 PM 推票时，用 Agent 系统性梳理投资逻辑：

```
使用投资逻辑Agent分析：
北方华创的投资逻辑是什么？
请从以下维度分析：
1. 行业beta（半导体设备景气周期位置）
2. 公司alpha（竞争优势、产品矩阵、客户结构）
3. 催化剂（近期订单、新品验证、政策推动）
4. 估值（当前PE/PB、历史分位、可比公司）
5. 风险（技术替代、客户集中度、海外制裁）
```

### 4.2 对比研究

做配对交易或选股时的对比分析：

```
批量查询：
1. 中际旭创 800G光模块 产能 客户结构
2. 新易盛 800G光模块 产能 客户结构
3. 光迅科技 数通光模块 市场份额
4. 光通信行业 毛利率对比 中际旭创 新易盛 光迅科技
5. 光模块公司 海外营收占比
```

接着用指标工具拉财务数据交叉验证：

```
查询 中际旭创 新易盛 光迅科技 最近4个季度营收和毛利率对比
```

### 4.3 催化剂日历梳理

为覆盖标的建立事件日历：

```
搜索最近30天半导体设备行业的公司公告，包括中标、订单、业绩预告、增发、股权激励
```

### 4.4 反向验证（挑战多头/空头逻辑）

在做出投资决策前，主动搜索反面证据：

```
搜索 半导体设备 风险 下行 去库存 订单放缓 产能过剩
```

```
搜索 光通信 竞争加剧 价格战 毛利率下滑
```


## Part 7 — 高级技巧与查询优化

### 7.1 资源类型过滤策略

不同的工作场景应该使用不同的资源类型过滤：

| 工作场景 | 推荐资源类型 | 原因 |
|----------|-------------|------|
| 了解卖方一致预期 | `券商研报` | 最权威的盈利预测来源 |
| 捕捉观点边际变化 | `分析师观点` | 比正式研报更及时 |
| 财报/增发/回购追踪 | `公司公告` | 一手信息，无偏差 |
| 产业链草根调研 | `调研纪要`、`会议纪要` | 获取非公开信息的最佳途径 |
| 政策与舆情感知 | `网络资源`、`产业公众号` | 覆盖面广，速度快 |
| 全面研究 | 不设过滤 | 初次覆盖时需要最大信息量 |

### 7.2 查询语句优化原则

**具体 > 模糊**

```
# 差
光模块市场

# 好
800G 1.6T光模块 数据中心 AI集群 需求量测算
```

**带公司名 > 纯行业**

```
# 差
半导体设备订单

# 好
北方华创 2025年 新增订单 中标公告 在手订单
```

**带时间 > 不带时间**

```
# 差
台积电资本开支

# 好
台积电 2025 2026年 资本开支计划 先进制程扩产
```

### 7.3 批量查询的拆分策略

将一个大问题拆成 5 个互补的子问题，覆盖不同信息维度：

```
主题：评估半导体设备板块投资价值

拆分为：
1. "半导体设备 行业景气度 订单增速 2025 2026"     → 需求侧
2. "半导体设备 国产化率 国产替代 进度"              → 竞争格局
3. "半导体设备 估值 PE PB 历史分位"                 → 估值水位
4. "半导体设备 风险 去库存 资本开支下修"             → 下行风险
5. "半导体设备 政策 大基金 产业扶持"                → 催化剂
```

### 7.4 多工具组合技

**"漏斗式"研究法**：从广到窄层层递进

```
Step 1: daily_report        → 发现信号（"盛美上海发布年报"）
Step 2: knowledge_search    → 拉取详细内容（限公司公告 + 券商研报）
Step 3: indicator           → 拉取结构化财务数据（营收/毛利率/订单）
Step 4: deep_research       → 形成投资判断（"盛美上海投资逻辑"）
```

**"交叉验证"法**：多信源对比

```
Step 1: knowledge_search(resource_types=["调研纪要"])    → 草根数据
Step 2: knowledge_search(resource_types=["券商研报"])     → 卖方观点
Step 3: knowledge_search(resource_types=["产业公众号"])   → 产业端视角
Step 4: indicator                                        → 用硬数据校验上述观点
```

**"对手盘思维"法**：主动寻找反面证据

```
# 如果你持多头，主动搜索空头逻辑
Step 1: knowledge_search("光通信 竞争加剧 价格战 毛利率压力")
Step 2: knowledge_search("光模块 库存 去化 需求放缓")
Step 3: deep_research("光通信板块当前面临的最大下行风险是什么？")
```
