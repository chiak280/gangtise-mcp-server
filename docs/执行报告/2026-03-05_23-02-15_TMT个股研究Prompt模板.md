# 执行报告：TMT 个股研究 Prompt 模板

**时间**：2026-03-05 23:02:15
**状态**：已完成

## 任务描述

为对冲基金 TMT 分析师定制一套标准化个股深度研究 Prompt 模板，满足以下需求：
- 模拟10年经验老牌分析师的研究框架
- 多维度分析（基本面、产业链、财务、竞争格局、券商观点、催化剂、风险）
- 在 TMT 板块内泛化，仅需替换股票名称即可使用
- 针对首次研究的陌生个股
- 输出专业分析师报告格式

## 设计思路

### 工具调用编排（9步）

| 步骤 | 工具 | 目的 |
|------|------|------|
| 1 | `gangtise_knowledge_batch` | 批量采集公司基础信息（5个子问题） |
| 2-3 | `gangtise_indicator` ×2 | 财务数据（营收利润 + 研发投入） |
| 4 | `gangtise_deep_research` (industry_expert) | 产业链全景分析 |
| 5 | `gangtise_knowledge_search` (券商研究报告) | 近90天券商深度报告 |
| 6 | `gangtise_knowledge_search` (首席分析师观点) | 近30天分析师观点 |
| 7 | `gangtise_knowledge_batch` | 近期催化剂采集（4个子问题） |
| 8 | `gangtise_knowledge_search` (纪要类) | 专家纪要和调研信息 |
| 9 | `gangtise_deep_research` (investment_logic) | 投资逻辑提炼 |

### 输出报告结构（9章）

1. 公司概况 → 2. 财务画像 → 3. 产业链定位 → 4. 竞争格局 → 5. 券商观点汇总 → 6. 近期催化剂 → 7. 核心风险 → 8. 投资逻辑总结（多空双面） → 9. 结论
